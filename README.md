# Product Carousel with Shopping Cart

A responsive React application featuring a product carousel and interactive shopping cart, built with TypeScript, Tailwind CSS, and Zustand for state management.

## Tech Stack

- **React 19.0.0** - UI library
- **TypeScript 5.9.3** - Type safety
- **Tailwind CSS 3.3.6** - Utility-first CSS framework
- **Zustand 5.0.8** - Lightweight state management
- **React Icons** - Icon library (Material Design icons)
- **React Slick 0.30.0** - Carousel component library

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd gameloft-task
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
